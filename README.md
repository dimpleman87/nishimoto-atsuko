# 西本敦子 公式サイト

島根県松江市を拠点に、防災食育講座・ローリングストック講座を届ける西本敦子さんの公式サイトです。

## 公開名

推奨するリポジトリ名・サイト名は `nishimoto-atsuko` です。

日本語URLは共有時に長いエンコード文字列になることがあるため、見た目の分かりやすさと管理しやすさを優先してローマ字表記にしています。

## ファイル構成

- `index.html`: サイト本体
- `assets/`: 本人写真、書籍画像、公式LINE QRコードなど
- `公開手順.md`: Netlify / GitHub Pages での公開手順

## 更新方法

1. `index.html` を編集します。
2. ブラウザで表示を確認します。
3. Gitで変更を記録します。

```powershell
git status
git add .
git commit -m "Update site"
```

## 注意

相談フォームはNetlify Forms向けに設定しています。GitHub Pagesだけで公開する場合、フォーム送信は動かないため、LINE・電話・メール導線を主導線として使います。
